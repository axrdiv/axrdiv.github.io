1. 几何技术

解决几何问题的关键是如何使特例数量尽可能的少，并且方便的实现算法。在这里将介绍几种解决集合问题方便的工具。

1.1 复数

复数形如，其中是一个虚数单位。复数的几何解释是一个二维坐标点或从原点指向的向量。

C++提供了complex类来解决几何问题。通过该类可以表示点或坐标，并对它们进行操作。首先可以定义一个坐标类型C。

typedef long long C;
typedef long double C;

之后可以定义一个复数类型P：

typedef complex<C> P;

最后通过宏定义x和y坐标

#define X real()
#define Y imag()

例如，定义点为：

P p = {4, 2};
cout << p.X << " " << p.Y << "\n";  // 4 2

P v = {3, 1};

Functions

complex类提供了一些能够用来计算几何问题的函数。如下的这些函数仅应在坐标类型为long double（或其他浮点类型）下使用。

函数abs(v)能够通过公式计算向量的模长。同时也能通过该函数计算两点之间的距离。

P a = {4, 2};
P b = {3, -1};
cout << abs(b-a) << "\n";   // 3.16228

函数arg(v)能够计算向量与轴之间的弧度。

函数polar(s, a)构造一个长度为s，角度为a的向量。通过将向量乘以一个长度为1角度为的向量来旋转度。

P v = {4, 2};
cout << arg(v) << "\n"; // 0.463648
v *= polar(1.0, 0.5);
cout << arg(v) << "\n"; // 0.963648

线段的交点

考虑求两个线段和的交点。如果两个线段存在交点，则有如下三种可能：

  1. 两个线段共线，且存在重叠部分。则具有无穷个交点。
  2. 两个线段具有一个公共顶点，且这是唯一的交点。
  3. 两个线段存在一个交点，且不为顶点。当每条线段的两个端点在另一条线段的两侧时，则为该情况。

2.3 凸包问题

凸包表示能够包裹给定点集的最小凸多边形。多边形的凸性是指多边形的任意两个顶点之间的连线完全在多边形的内部。

有很多算法能够高效的构建凸包。其中最简便的应当是Andrew’s Algorithm。算法首先找出最左和最右的点，之后先构造凸包的上边，随后构造凸包的下边。构造上下边的方法相似，因此这里只介绍如何构造上边界。

首先对所有点按照主坐标值，副坐标值进行排序。之后遍历这些点，依次加入一个表示凸包的栈。在加入一个点后，需要确保与栈顶坐标点新构成的边不向左转，否则不断从凸包栈中移除栈顶的点，直至新的边不向左转。下图展示了该构建过程。
